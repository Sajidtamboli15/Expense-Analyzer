# Expense Analyzer

A Python web application to track and analyze expenses with machine learning categorization and interactive charts.

## Features
- User signup/login system
- Add, delete, and view expenses
- CSV upload with auto-categorization
- Machine learning model for category prediction
- Interactive charts (Pie & Bar) using Plotly

## Tech Stack
- Python (Flask, Pandas)
- MySQL
- Plotly
- HTML & CSS
